# 🏏 Pandavas Cricket Club Scorer v3.1

Complete Production-Ready PWA for cricket match scoring with Black + Gold premium theme.

## Features
- ✅ Live Scoring with all cricket actions (0-6, Wide, No Ball, Wicket, Run Out, Retired Out)
- ✅ Full Cricket Engine (Powerplay, Free Hit, DLS, Super Over)
- ✅ Team & Player Management with photos and stats
- ✅ Tournament Module with Points Table, NRR, Fixtures
- ✅ Leaderboards (Orange Cap, Purple Cap, MVP)
- ✅ Auto-Generated Match Reports with MOM
- ✅ PWA Support (Installable on iOS/Android)
- ✅ Firebase Integration (Auth, Firestore, Hosting)
- ✅ Real-time Sync with Offline Queue
- ✅ Wagon Wheel & Manhattan Graph
- ✅ Auto Backup & Export/Import
- ✅ Live Score Sharing & QR Code
- ✅ Black + Gold Premium UI Theme

## Tech Stack
- HTML/CSS/JavaScript (Vanilla)
- Firebase (Auth, Firestore, Hosting, Storage)
- PWA (Service Worker, Manifest)
- QRCode.js, html2pdf.js
